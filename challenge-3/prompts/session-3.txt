2. **fact-checker agent + script** (LLM-free):
   - `aiwriting/agents/aiwriting-fact-checker.md` — agent 본문은 "scripts/fact_checker.py 호출" 안내만
   - `scripts/fact_checker.py <md_path> --known known_facts.yml`
   - `scripts/fact_checker_patterns.py` — 5 type regex (PRD §3 D3)
     - (a) 숫자: `\b\d+(\.\d+)?\b`, `\b\d+(\.\d+)?%\b`, `[\$₩€]\d+(\.\d+)?[KMB]?\b`

     - (c) 직접 인용: `"[^"]{8,}"`, `「[^」]{8,}」`
     - (d) 날짜: `\b\d{4}\b`, `\b\d{4}-\d{2}\b`, `\b\d{4}-\d{2}-\d{2}\b`
     - (e) 고유명사: yaml `proper_nouns_known: [...]` list 외
